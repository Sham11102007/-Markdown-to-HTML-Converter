# Markdown to HTML Converter

## Description

Markdown to HTML Converter is a Python-based application designed to convert Markdown-formatted text into HTML code. The project supports basic Markdown elements such as headings and bold text, allowing users to transform formatted text into web-compatible HTML output through a simple command-line interface.

This project demonstrates the practical use of Python concepts such as string manipulation, functions, conditional statements, and user input handling. It serves as a beginner-friendly text conversion utility and provides a foundation for understanding how markup languages can be processed for web development.

## Features

- Convert Markdown headings into HTML heading tags
- Convert Markdown bold text into HTML bold tags
- Simple command-line interface
- Fast and beginner-friendly text conversion
- Easy to understand and implement

## How to Run

1. Open the project in PyCharm or any Python IDE.
2. Run the `main.py` file.
3. Enter Markdown text when prompted.
4. View the converted HTML output.
